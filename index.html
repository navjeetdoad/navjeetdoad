A-Star Algorithm Visualizer

A visualization of the A Star Pathfinding algorithm made in Python. This allows the user to plot a start point, an end point, and barriers between the 2 points, and the program computes the shortest path between the 2 points. Visit the GitHub repository for more information!

Graphical Application

A graphical design application reminiscint of MS Paint which includes various features, such as undo/redo, drawing, polygons, stickers, etc. Made using Python, along with the Pygame graphics framework. This application was for a project and it required a theme, so the theme is the notorious anime Dragon Ball Z. Visit the GitHub repository for more information!

Binding of Isaac: Rebirthed Again

A recreation of the famous top-down dungeon-crawling game Binding Of Isaac. The game was made in Python using the Pygame framework, along with the Math framework in order to create in-game physics. Visit the GitHub repository for more information!

Text-Based Pokemon Battle Simulator

A Java text based creation of a Pokemon battle simulator created with Object Oriented Programming. The program reads Pokemon from a text file, turns them into objects and allows the user to play against the computer with their selection of Pokemon. Visit the GitHub repository for more information!

2048

A recreation of the notorious game 2048. Created using various Object Oriented Programming techniques, along with the LibGDX library in Java for a project for the ICS4U course in high school. Visit the GitHub repository for more information!
